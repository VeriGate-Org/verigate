Material-standard easing. Fast start with a gentle deceleration, making interactions feel snappy yet smooth.

Motion Rules:

  • Always use the standard easing curve — never linear or ease-in-out
  • Prefer transform and opacity for animations (GPU-accelerated)
  • Respect prefers-reduced-motion — disable non-essential animations
  • Scroll-triggered animations: fade-in with y:20 → 0, stagger children by 100ms

12 — Do's & Don'ts

Do's & Don'ts

Quick-reference rules for maintaining brand consistency across all VeriGate touchpoints.

Do's

Use approved background colours — white, off-white, navy, or dark navy for the logo.

Maintain clear space — equal to the shield height on all sides of the logo.

Use Inter for all body text — weights 300–700 as specified in the type scale.

Reserve cyan for accents — links, icons, highlights, and interactive states only.

Use the 8px spacing scale — all margins, padding, and gaps should be multiples of 8 (or 2/4 for fine-tuning).

Respect the elevation hierarchy — Base → Raised → Sticky → Modal for consistent depth.

Don'ts

Don't stretch or distort the logo — always scale proportionally, never warp.

Don't recolour the shield — the shield is always red (#E23D36) with a white checkmark.

Don't use Manrope for body text — Manrope is exclusively for the wordmark. Use Inter everywhere else.

Don't place the logo on busy backgrounds — no photos, patterns, or gradients without sufficient contrast.

Don't use red as a general accent — red is reserved for the shield, destructive actions, and urgent states.

Don't mix website and portal styles — each design language serves a distinct audience and context.
